# ATC Management System

A comprehensive Air Traffic Control management system with Java backend API and Java Swing frontend.

## Features

- **System-Driven Flight Generation**: Flights are automatically generated by the simulator, not manually entered.
- **Controller-Centric Design**: Controllers manage assigned flights through a dashboard.
- **Multi-User Authentication**: Secure login for controllers with role-based access.
- **Real-Time Updates**: Dashboard refreshes automatically to show current flight statuses.
- **Emergency Handling**: Priority-based emergency management with automatic scheduling adjustments.
- **Resource Assignment**: Assign runways, gates, and update flight statuses.

## Architecture

- **Backend**: Java with JDBC, REST API using HttpServer
- **Frontend**: Java Swing desktop application
- **Database**: MySQL with stored procedures and triggers
- **Simulator**: Background thread generating flights every 15 seconds

## Setup

1. Set up the database:
   - Run `database/schema.sql`
   - Run `database/procedures.sql`
   - Run `database/triggers.sql`
   - Run `database/seed.sql`

2. Configure database connection in `src/db/DBConfig.java`

3. Compile and run the backend:

   ```
   cd backend
   run.bat
   ```

4. Run the frontend:
   ```
   cd frontend
   run_frontend.bat
   ```

## API Endpoints

- `POST /login` - Authenticate controller
- `GET /flights` - Get assigned flights
- `GET /flights/{id}` - Get flight details
- `POST /flights/emergency` - Declare emergency
- `POST /flights/runway` - Assign runway
- `POST /flights/gate` - Assign gate

## Usage

1. Start the backend server
2. Launch the frontend application
3. Login as a controller (ID 1, 2, or 3)
4. View assigned flights in the list
5. Select a flight and click "View Details" to manage it
6. Use action buttons to declare emergencies, assign resources, etc.
7. The system will automatically generate new flights and assign them to controllers
